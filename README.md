# 🏠 House Price Prediction

## 📌 Project Overview

This project was completed as part of an internship assignment focused on predicting house prices using Machine Learning techniques. The objective was to analyze housing data, identify the factors that influence property prices, and build predictive models capable of estimating house values based on various property features.

---

## 📊 Dataset

**Housing Prices Dataset**

🔗 Source: https://www.kaggle.com/datasets/yasserh/housing-prices-dataset

The dataset contains information about residential properties along with their corresponding selling prices.

---

## 🛠️ Technologies & Libraries Used

- 🐍 Python 3
- 📓 Jupyter Notebook
- 🐼 Pandas
- 🔢 NumPy
- 🤖 Scikit-learn
- 📈 Matplotlib
- 🎨 Seaborn

---

## 💡 Key Insights

- 📏 Property area showed a strong influence on house prices.
- 🏡 Features such as bathrooms, parking spaces, and furnishing status contributed significantly to price variation.
- 🌲 Random Forest Regressor generally produced more accurate predictions than Linear Regression.
- 📊 Data visualization revealed meaningful correlations between housing attributes and property value.
